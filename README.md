
# 多角形（点・辺）判定Webアプリ
<img width="536" height="497" alt="スクリーンショット 2026-06-13 102534" src="https://github.com/user-attachments/assets/3260ad80-05df-4d9c-bd4b-26fdbe6c0629" />

画像をアップロードするだけで、描かれている図形の「頂点（点）の数」と「辺の数」を自動で解析し、何角形かをインテリジェントに判定するWebアプリケーションです。

##  主な機能
* **画像の自動解析**: ドラッグ＆ドロップまたはファイル選択で画像をアップロード可能。
* **図形認識（OpenCV.js）**: コンピュータビジョン技術を用いて、画像内の最も大きな図形を自動検出。
* **多角形近似（点・辺のカウント）**: 輪郭のノイズを除去し、正確な頂点数・辺の数を算出。
* **視覚的フィードバック**: 検出した図形の輪郭を赤い線でハイライト表示。

## 🛠 テクノロジー・スタック
* **フロントエンド**: HTML5, CSS3, JavaScript (Vanilla JS)
* **画像処理コア**: OpenCV.js (v4.5.4)
* **バックエンド / サーバー**: Node.js (Express)

## 🚀 検出の仕組み（アルゴリズム）
本アプリはOpenCVを利用し、以下のステップで画像処理を行っています。

1. **グレースケール化 & ぼかし**: 画像の色情報を破棄し、輪郭検出の邪魔になる微細なノイズを平滑化します。
2. **二値化（Otsuの自動しきい値選定）**: 背景と図形を白黒2色に完全に分離します。
3. **輪郭抽出（findContours）**: 画像内のオブジェクトの外縁線を検出します。
4. **多角形近似（Douglas-Peucker アルゴリズム）**: 
   検出したギザギザの輪郭線を、設定された許容誤差（周長の4%）の範囲内で「直線」へと近似し、最終的な頂点の数を割り出します。

💡 判定精度を上げるためのコツ
はっきりした画像: 「白背景に黒い図形」など、背景と図形のコントラストが高い画像を使用すると精度が100%に近づきます。

デジタル描画: 手書きの線はうねりがあるため、余計な「点」としてカウントされやすくなります。ペイントソフトなどで直線的に描かれた図形が最適です。

感度調整: 手書き画像などで判定がズレる場合は、index.html 内の 0.04 * peri（許容誤差）の数値を 0.05〜0.07 などに調整することで、より直線的に（大雑把に）判定させることができます。
